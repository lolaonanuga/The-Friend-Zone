# The Friend Zone

The Friend Zone is a Ruby on Rails web app to find new friends with common interests.



## Installing

Git clone the repo and follow the steps below

```
$ bundle

```

# Setting up the environment
```
$ rails db create && rails db migrate
$ rails db seed
$ rails s
```
